# 🇻🇳 AIDEOM-VN

**AI-Driven Decision Optimization Model for Vietnam**

Web app giải 12 bài toán mô hình ra quyết định phát triển kinh tế Việt Nam trong kỉ nguyên AI.

## Chạy app

```bash
pip install -r requirements.txt
streamlit run Home.py
```

## Cấu trúc

- `Home.py` — Trang chủ tổng quan
- `pages/` — 12 bài toán theo 4 cấp độ
- `*.csv` — Dữ liệu thực Việt Nam 2020-2025
